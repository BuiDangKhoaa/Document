## Hosting
Hosting hay web hosting là không gian lưu trữ dữ liệu được chia nhỏ từ các máy chủ (server), giúp đăng tải dữ liệu và xuất bản website, app trên Internet. Khi đăng ký dịch vụ hosting, nghĩa là bạn đang thuê một chỗ đặt chứa tất cả các file và dữ liệu cần thiết lên trên server để website có thể hoạt động được 24/7.
